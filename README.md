## 📖 ゲーム概要

**グラムちゃん**（紫銀の髪・異色瞳・天使の翼・クリスタルハートの女の子）を操作して、  
降り注ぐ敵（スライム・コウモリ・ダークスライム）を撃ち倒す**2Dピクセルアートシューティングアクション**。

- 📱 **スマホ縦持ち** → 上から降ってくる敵を撃ち落とす
- 🖥️ **スマホ横持ち・PC** → 左右から現れる敵を迎撃
- 🎨 **全スプライトAI生成**（SDXL + Pixel Art XL LoRA）
- 🔫 **モダンタッチ操作**（コントローラーUIなし）

---

## 🎮 遊び方

### 操作方法（タッチパネル全体がコントローラー！）

| 操作 | 動作 |
|------|------|
| 👆 **左右にスライド** | グラムちゃんがその方向にスムーズに移動 |
| ⬆️ **上にフリック** | ジャンプ |
| 👆 **画面をタップ** | ジャンプ（短いタップ） |
| 🔫 **画面を長押し（0.2秒以上）** | **自動連射！** 指を置いたままで勝手に撃つ |

### 🎯 ゲームルール

1. **敵を撃ってスコアゲット！**
   - スライム（緑）→ 10pt
   - コウモリ（紫）→ 15pt
   - ダークスライム（黒紫）→ 20pt

2. **コンボシステム**
   - 連続で敵を倒すと **COMBO × スコア倍増！**
   - COMBO x3以上でスコア色が金色に変化

3. **アイテム回収**
   - ⭐ **星** → 50pt獲得
   - ❤️ **ハート** → HP回復（最大5まで）

4. **ステージ進行**
   - 200ptごとに **STAGE UP！**
   - ステージが上がると敵の出現頻度アップ

5. **GAME OVER条件**
   - 敵にぶつかるとHP-1
   - HP（❤️）が0になるとゲームオーバー
   - ハイスコアを記録してリトライ！
