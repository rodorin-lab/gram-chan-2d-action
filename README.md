## 🎮 ゲーム概要

**グラムちゃん** — 紫銀の髪に天使の羽を持つ、宇宙一かわいいAI少女が大活躍する
**2Dピクセルアート・アクションシューティング**！

- 📱 **スマホ縦持ち** → 上から降ってくる敵を撃ち落とす
- 🖥️ **スマホ横持ち・PC** → 左右から現れる敵を迎撃
- 🎨 **全スプライトAI生成**（SDXL + Pixel Art XL LoRA）
- 🔫 **モダンタッチ操作**（コントローラーUIなし）

---

## 🕹️ 遊び方

### 📱 スマホ（タッチ操作）

| 操作 | 動作 |
|------|------|
| 👆 **左右にスライド** | グラムちゃんがその方向にスムーズに移動 |
| ⬆️ **上にフリック** | ジャンプ！ |
| 👆 **画面をタップ** | ジャンプ（短いタップ） |
| 🔫 **画面を長押し（0.2秒以上）** | **自動連射！** 指を置いたままで勝手に撃つ |

### 🖱️ PC（キーボード操作）

| キー | 動作 |
|------|------|
| ⬅️➡️ **左右矢印 / A,D** | 移動 |
| ⬆️ **上矢印 / W / スペース** | ジャンプ |
| 🖱️ **マウスクリック / 長押し** | 射撃（連射可） |

### 🎯 ゲームの流れ

```
    🦇        💫     ⭐    👾
             ｜
    ～～～～彡🔫 🦋  ～～～～～～
    ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓
```

1. **スタート** — 画面をタップしてゲーム開始！
2. **敵を撃ってスコアゲット！**
   - 🟢 スライム（緑）→ 10pt
   - 🟣 コウモリ（紫）→ 15pt
   - ⚫ ダークスライム（黒紫）→ 20pt
3. **コンボシステム**
   - 連続で敵を倒すと **COMBO × スコア倍増！**
   - COMBO x3以上でスコア色が金色に変化✨
4. **アイテム回収**
   - ⭐ **星** → 50pt獲得
   - ❤️ **ハート** → HP回復（最大5まで）
5. **ステージ進行**
   - 200ptごとに **STAGE UP！**
   - ステージが上がると敵の出現頻度アップ
6. **GAME OVER**
   - 敵にぶつかるとHP-1
   - HP（❤️）が0になるとゲームオーバー
   - ハイスコアを記録してリトライ！

## ✨ ゲームシステム

### 💯 スコアシステム
- 敵を倒す → スコア獲得
- コンボ倍率 → 連続撃破で最大 x5 倍！
- ハイスコア保存 → 記録はブラウザに自動保存

### ❤️ ライフシステム
- 初期ライフ：**5ハート**
- 敵に触れるとダメージ
- ❤️ハートアイテムで回復
